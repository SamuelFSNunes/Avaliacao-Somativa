POO e Java

  Com ajuda de um curso proporcionado pela empresa ORACLE, demos inicio a POO (Programação Orientada a Objeto), sendo assim aprendemos conceito de classes, objetos, métodos, construtores, herança, abstração, encapsulamento, polimorfismo e tratamento de erros.

Para consolidar o conhecimento, trabalhamos em exercícios lógicos que envolviam exibição dessas arrays, cálculos, identificação de diagonais primárias e secundárias, bem como manipulação de linhas e colunas.

Banco de Dados

Inicialmente, fomos apresentados aos conceitos de banco de dados, que podem ser divididos em dois tipos: relacionais e não relacionais. Diante disso, utilizamos o SGBD "PostegreSQL" como nossa ferramenta de gerenciamento.

Os bancos de dados relacionais são organizados por meio de modelos como o MER (Modelo de Entidade-Relacionamento) e o DER (Diagrama de Entidade-Relacionamento). Esses modelos nos ajudam a dividir os dados em entidades, atributos e relacionamentos, facilitando a interpretação e compreensão do banco de dados. Sabendo disso, fomos divididos em Squads e tivemos como desafio criar um modelo MER, DER e um banco de dados com base nas orientações recebidas.

Durante a criação do banco de dados, utilizamos comandos DDL (Data Definition Language) para criar, alterar e excluir tabelas e colunas. Além disso, utilizamos comandos DML (Data Manipulation Language) para popular essas tabelas e, por fim, utilizamos o DQL (Data Query Language) para realizar consultas nesse banco de dados.

Após a criação do banco de dados, aprendemos sobre as três formas de normalização e aplicamos esses conceitos na tabela TITANIC. Todo esse trabalho foi realizado em um ambiente real de banco de dados fornecido pelo SENAI, onde aprendemos a importância de evitar comandos como "DELETE sem WHERE", que podem causar consequências indesejadas no banco de dados.

Tinkercad

  Com uma compreensão abrangente desses conceitos, avançamos para o estudo de uma nova linguagem de programação: o C++. Essa linguagem de baixo nível é amplamente utilizada no desenvolvimento de softwares e dispositivos embarcados. Aprendemos sua sintaxe e aplicamos nossos conhecimentos diretamente no TINKERCAD, uma plataforma online que nos permite simular automações industriais e aplicações com arduino.

  Ao utilizar essa plataforma, podemos criar e simular circuitos eletrônicos, desenvolver protótipos de produtos e explorar conceitos de automação industrial de maneira intuitiva. Essa plataforma oferece uma variedade de componentes eletrônicos virtuais, como microcontroladores, sensores e atuadores, que podem ser conectados e programados para criar projetos funcionais.
